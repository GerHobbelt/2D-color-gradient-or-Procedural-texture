How the image is created?
x,y : integer ( coordinates of pixel) --> k : integer ( index of color) --> color:TColor ( color of the pixel)
--> RGB ( color of pixel)
how to add new function ?
  1. in unit FunctionsM add new function type to definition of TFunctionType
  2. in form OptionDlg in FunctionsGroup add a new radioButton ( in object inspector open FunctionsGroup and then klik on items to see the editor; add new string)
  3. in optionsDlgM in procedure TOptionsDlg.OKButtonClick add new line
  4. you also may changen kMax in Procedure Setup in unit FunctionsM ( add new line if there is no such line in case statement, it means that else is used

Sinus: color:=sin(x)
------------------------
result:=y+round(height*(sin(DegToRad(x))+1)/2);
value of sin has a range [-1,+1]
(sin+1) [0,2]
(sin+1)/2 [0,1]
If you want to see more periods of sin add:
result:=y+round(height*sin(2*Pi*DegToRad(x))+1)/2);
If you want to see sin more flat:
result:=y+round(height*sin(2*Pi*DegToRad(x))+1)/8);
